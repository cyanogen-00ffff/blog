Took me a while to figure things out; the reason why the entries under the new directory didn't look the way I want to (title as header, links of previous and text post underneath) is because the 11tydata.js file, which calls for the post layout, and I'm guessing also puts the tag 'posts' as well, doesn't have the correct directory prefix. It worked under /blog *because* it's named blog.11tydata.js, so if it's under /entries it should be named entries.11tydata.js.
